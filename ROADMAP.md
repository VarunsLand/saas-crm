# Project Roadmap

## Completed Features
- [x] Multi-tenant Authentication System
- [x] CRM Pipeline (Kanban Drag & Drop)
- [x] Lead & Customer Entities separation
- [x] Financial Tracking (Revenue & Expenses)
- [x] KPI Dashboard & Charts
- [x] Universal Notes / Interactions System
- [x] Task Management & Action Center
- [x] Global Search (Cmd+K)
- [x] Customer 360° Journey Timeline
- [x] Quick Action FAB
- [x] Empty States & UI Polish

## In Progress
- [ ] Role-Based Access Control (Admin vs Staff permissions refinement)
- [ ] Responsive UI refinements for mobile devices
- [ ] Webhook architectures for external integrations

## Future Features
- **Email Integration:** Two-way sync with Gmail/Outlook to log emails directly into the Customer Timeline.
- **Push Notifications:** In-app and browser notifications for Overdue Tasks and Lead assignments.
- **Advanced Reports:** Custom report builder allowing users to export PDF and CSV data cuts.
- **AI Features:** 
  - AI Lead Scoring based on interaction frequency.
  - Generative AI email drafts inside the Notes system.
  - Anomaly detection for unusual expense spikes.
